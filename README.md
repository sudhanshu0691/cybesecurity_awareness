# Cyber Suraksha Conclave 2026

A cybersecurity awareness event website designed with modern web technologies and cutting-edge UI/UX.

## 🌐 Live Demo
https://sudhanshu0691.github.io/cybesecurity_awareness/

## 🚀 Features
- Modern React + TypeScript + Vite stack
- Beautiful animations with Framer Motion
- Bilingual support (English & Hindi)
- Responsive design with Tailwind CSS
- Dark theme with cyber-inspired aesthetics
- Live speaker information
- Career opportunities section
- Dynamic impact metrics

## 📋 Tech Stack
- **Frontend**: React 19, TypeScript
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Hosting**: GitHub Pages

## 🛠️ Development

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation
```bash
npm install
```

### Development Server
```bash
npm run dev
```
Server runs on `http://localhost:3000`

### Build for Production
```bash
npm run build
```

### Type Checking
```bash
npm run lint
```
